HTML Part 1

요소(Element)

HTML 문서를 구성하는 항목. HTML은 요소들의 집합이고 요소는 다른 요소를 포함할 수 있다.

태그(Tag)

요소를 작성할 때, 요소의 시작과 끝을 나타내고 요소의 유형을 지정하기 위해 작성하는 표식.

태그는 여는 태그와 닫는 태그가 있으며 부등호 안에 태그 이름을 넣어 작성하며, 닫는 태그의 경우, 여는 부등호와 태그 이름 사이에 슬래시(/)를 붙여서 작성한다.

위 코드에서 <div>텍스트</div>가 하나의 요소이고, <div>, </div>가 태그, <div>가 여는 태그, </div>가 닫는 태그다.

문단(Paragraph) 태그(p 태그) 사용하기

문단 태그를 사용하여 문단을 구분해주면 문단 단위로 줄바꿈이 실행된다.

제목(Heading) 태그(hn 태그)

제목 태그는 문서의 제목을 작성할 때 사용하며 h1부터 h6까지 6단계로 하위 항목을 작성할 수 있다. 자세한 것은 코드 예시를 확인하자.

위 코드 예시를 보면, "대한민국 헌법"이라는 항목 안에 "제1조"와 "제2조"라는 하위항목이 제목 태그(h1, h2)로 작성되어 있다.

리스트 태그

리스트는 순서없는 리스트(ul, Unordered List)와 순서있는 리스트(ol, Ordered List)로 나뉜다. 각각 ul, ol 태그를 통해서 작성하며 리스트의 항목은 li 태그(List Item)를 통해 작성한다.

하위 리스트

리스트 안에 하위 리스트를 넣을 수도 있다.

이미지 태그(img 태그)

이미지 태그는 이미지를 삽입할 때 사용한다. src 속성에 이미지 파일의 URL을 입력하고, width, height 속성을 통해 크기를 지정해줄 수 있다.

하이퍼링크

월드 와이드 웹(WWW)는 페이지들이 서로 연결되어 클릭을 통해 이동할 수 있도록 구상되었다. 이러한 연결을 하이퍼링크라고 한다.

HTML에서 하이퍼링크는 a 태그를 통해 구현할 수 있다. a 태그의 href 속성에 URL을 지정해주면, 링크 클릭 시 지정된 URL로 이동된다.
