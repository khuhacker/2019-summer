CSS

CSS란?

HTML 요소가 실제로 화면에 어떻게 표시될 지를 기술하는 언어.

CSS 문법

CSS는 다음과 같은 문법으로 작성된다.

HTML 문서에 CSS 적용하기

style 태그 안에 작성하기

CSS 파일 사용하기

선택자(Selector)

선택자란 CSS에서 HTML 요소들 중, 스타일을 적용할 요소를 선택해주는 문구이다.

요소 선택자

요소 선택자는 해당 유형의 요소를 전부 선택하는 선택자로, 태그 이름이 선택자가 된다.

ID 선택자 사용하기

요소의 id 속성을 이용하여 요소를 선택하는 선택자로, # 뒤에 요소의 id 값을 적어주어 작성한다.

HTML 문서를 작성할 때, id 속성이 다른 요소와 중복되게 작성하면 안되므로, ID 선택자를 사용하면, 단 하나의 요소를 정확하게 선택할 수 있다.

클래스 선택자 사용하기

요소의 class 속성을 이용하여 요소를 선택하는 선택자로, . 뒤에 선택할 class 값을 적어주어 작성한다.

id 속성과 다르게 class 속성은 다른 요소와 같은 class를 사용할 수 있고, 한 요소가 여러 개의 class 값을 가질 수 있다. 이 경우, 각 class 이름을 공백으로 구분하여 작성한다.

가상 클래스(Pseudo-class) 선택자

가상 클래스는 요소가 특정 상태인 경우에만 선택하는 선택자이다.

가상 클래스 선택자는 주로 일반 선택자 뒤에 이어서 작성한다.

:nth-child 선택자

:nth-child 선택자는 부모의 몇번째 자식인지로 선택 여부를 결정한다. :nth-child(n) 형식으로 사용하며, n은 정수를 입력할 수 도 있고, n에 대한 식을 입력할 수 도 있다. 정수를 입력할 경우 n번째 자식인 요소를 선택하고, n에 대한 식을 입력한 경우, n에 모든 정수를 대입했을 때 식의 결과 k에 대하여, k번째 자식을 선택한다. 예를 들어 :nth-child(2n)은 짝수번째 요소,:nth-child(2n+1)은 홀수번째 요소를 선택한다.

:first-child, :last-child 선택자

:first-child, :last-child 선택자는 각각 부모 요소의 첫번째/마지막 자식인 요소를 선택한다.

:not 선택자

:not 선택자는 선택자로 선택한 요소 중, 특정한 선택자로 선택할 수 있는 요소를 제외하는 선택자다.

:hover 선택자

:hover 선택자는 마우스를 올린 요소를 선택하는 선택자이다.

속성(Property)

font-family 속성

글꼴을 설정한다.

font-weight 속성

글씨의 굵기를 설정한다.

normal: 보통(기본)

bold: 굵게

text-align 속성

글씨의 정렬 방향을 설정한다.

color 속성

텍스트의 색상을 지정해주는 속성이다.

색상 이름을 통해 지정하는 방법과, 색상 코드를 통해 지정하는 방법이 있다.

background-color 속성

요소의 배경색을 지정하는 속성이다.

color 속성과 마찬가지로 색상 이름이나 색상 코드를 통해 색상을 지정할 수 있다.

width, height 속성

width, height 속성은 각각 block 요소의 가로, 세로 크기를 지정한다.

주로 사용하는 단위로는 px(픽셀)과 %가 있다.

%의 경우 부모 요소의 크기에 비례해서 설정된다.

margin 속성

요소의 바깥 여백을 설정한다.

상하좌우 여백을 서로 다르게 줄 경우, margin-top, margin-right, margin-bottom, margin-left로 나누어 지정하거나, 4개를 한 번에 지정 가능한데 이 경우, 상단부터 시작하여 시계방향으로 지정한다.

padding 속성

요소의 안쪽 여백을 설정한다.

상하좌우 여백을 서로 다르게 줄 경우, margin-top, margin-right, margin-bottom, margin-left로 나누어 지정하거나, 4개를 한 번에 지정 가능한데 이 경우, 상단부터 시작하여 시계방향으로 지정한다.

position 속성

position 속성은 요소의 위치를 설정해주는 방법을 지정한다.

position 속성을 지정해준 뒤, top, left, bottom, right 속성을 이용하여 위치를 지정해줄 수 있다.

position: static

position 속성의 기본 값으로, 원래 위치 그대로 지정된다.

position: absolute

조상 요소 중, position이 static이 아닌 가장 가까운 조상의 위치를 기준으로 위치가 설정된다.

position: relative

요소의 원래 위치에서 상대적으로 위치를 이동시킨다.

position: fixed

요소를 스크롤에 상관 없이 화면 기준 위치에 고정시킨다.
